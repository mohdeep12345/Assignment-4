About

Welcome to our web application! This application is built as part of the WEB322 course assignment. It showcases an online store with various items and categories.

Here, you can browse through different items, view details about each item, and add new items to the store.

This project demonstrates the use of Node.js, Express, and Cloudinary for managing image uploads.
